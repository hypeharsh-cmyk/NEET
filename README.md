# NEET Biology — Mock Mastery 50

A single, self-contained, **mobile-first** HTML question bank: 50 NEET Biology MCQs
drawn from real Aakash mock papers (NEET-2023 & 2025), re-typeset with full
explanations, memory tips, and "why it was tricky" trap-busters.

👉 **Open [`neet_biology_qbank_mock2.html`](neet_biology_qbank_mock2.html)** in any
browser (double-click, no server needed).

## Features
- **50 questions** across 10 chapters — Genetics & Evolution, Human Physiology,
  Reproduction, Plant Physiology, Cell Biology & Biomolecules, Diversity of Life,
  Morphology & Anatomy, Ecology, Biotechnology, Microbes & Human Health.
- **13 hand-drawn SVG diagrams** (meiosis, DNA replication fork, transcription unit,
  lac operon, cardiac conduction, menstrual cycle, cyclic photophosphorylation,
  Krebs substrate-level phosphorylation, endarch/exarch xylem, cell cycle, pBR322
  map, root Casparian strip) — every figure's labels are referenced by its question.
- Every question carries a **memory tip** and a **trap-buster** explaining the exact
  NCERT detail being tested.
- NEET marking (**+4 / −1**), no timer, full results analysis: motivation card,
  chapter-wise performance, and a complete mistake review.
- Click-to-zoom figures, MathJax-typeset formulae, white-&-pink theme, thumb-first
  navigation with a question palette.

*Practice tool for personal study use. Question content is adapted from Aakash NEET
mock papers; the source PDFs are not distributed here.*
